CRITICAL NOTE v1.2: This matrix measures CO-OCCURRENCE only. It does NOT establish:
• Causal direction (which function "causes" or "requires" another)
• Logical necessity (whether functions are structurally dependent)
• Derivational structure (whether one function can be derived from another)
To evaluate H3 fully, additional evidence is needed: qualitative coding, logical analysis, structural derivation.

CODING PROTOCOL
1. Read the full text (not just abstract) before scoring.
2. Score what the work DEMONSTRATES, not what it CLAIMS.
3. If a work scores N on a function, use the "Why not N+1?" column to justify.
4. If uncertain between two scores, use the LOWER score (conservative).
5. Document the evidence (page number, quote) for every score of 2 or 3.
6. Two independent coders required for scores ≥2. Compute Cohen's kappa.
7. Disagreements resolved by discussion; if unresolved, use third coder.
8. All coding decisions are recorded in the Decision Log with evidence source.

Research Purpose

This review matrix is designed to test whether existing literature provides an operational, cross-sectoral governance framework capable of observing, interpreting, and acting upon threshold dynamics before systemic conditions become materially irreversible.

RQ1: What functions must an operational, cross-sectoral governance framework perform in order to act on threshold dynamics before irreversibility, according to the existing literature?

RQ2: To what extent do existing frameworks individually satisfy those functions?

Preliminary Interpretation

The preliminary screening suggests a fragmentation pattern: financial macroprudential frameworks are operational but sector-specific, while cross-sectoral and threshold-oriented frameworks remain weaker on executable decision protocols and reversibility.

Among the corpus, only one entry is classified as an operational system. Coverage claims should therefore be reported within type classes rather than across the full set.
